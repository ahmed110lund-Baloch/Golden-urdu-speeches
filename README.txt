Golden Speeches - PWA-ready package
====================================

Files:
- index.html        -> app index (fہرستِ تقاریر)
- speech27.html
- speech28.html
- style.css
- manifest.json
- service-worker.js
- icons/icon-512x512.png
- icons/icon-192x192.png

How to use:
1. Unzip this package and upload the entire folder to Netlify (drag & drop in Sites).
2. Ensure files are served over HTTPS (Netlify does this by default).
3. Open the site on a mobile browser (Chrome/Edge). You should see 'Install' / 'Add to Home Screen'.
4. If you want more speeches, add them as speechXX.html and link in index.html.
